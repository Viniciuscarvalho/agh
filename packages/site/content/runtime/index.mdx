AGH Runtime docs are organized by the job you are trying to do right now: get AGH running on one
machine, understand how the runtime behaves, or inspect the exact contract of one operator surface.

<OperatorNote>
  New to AGH? Start with Installation, then Quick Start. Come back to Core Concepts when you want
  the operating model, and use CLI Reference or API Reference when you need exact contracts.
</OperatorNote>
